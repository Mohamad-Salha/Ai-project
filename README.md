# 🧩 AI Maze Solver

An interactive maze generation and solving visualization tool that implements **Uniform Cost Search (UCS)** and other search algorithms to find optimal paths through randomly generated mazes.

> 🎓 **University Project** — Artificial Intelligence Course | An-Najah National University

---

## 🎯 Project Overview

This project demonstrates fundamental AI search algorithms by visually solving mazes in real-time. Users can generate random mazes and watch the AI algorithm explore and find the optimal path from start to finish.

---

## 💻 Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Algorithms:** Uniform Cost Search (UCS), Search Algorithms
- **Visualization:** Custom JavaScript canvas/DOM rendering

---

## ✨ Features

- ✅ **Random Maze Generation** — Generate unique mazes each time
- ✅ **AI Path Finding** — Implements UCS to find the optimal path
- ✅ **Visual Step-by-Step Solving** — Watch the algorithm explore the maze in real-time
- ✅ **Interactive UI** — Custom-designed interface for maze interaction
- ✅ **Responsive Design** — Works in modern web browsers

## 🤖 How It Works

1. **Maze Generation** — A random maze grid is generated with walls and open paths
2. **Start & Goal** — Starting position and goal position are defined
3. **UCS Algorithm** — Uniform Cost Search explores the maze, expanding nodes by lowest path cost
4. **Visualization** — Each step of the algorithm is rendered visually so you can see the exploration process
5. **Solution Path** — Once the goal is found, the optimal path is highlighted
